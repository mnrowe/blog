Welcome. I'm Matthew — a Senior Information Security Officer working across security operations, cloud and identity, and automation.

Day to day I run detection and response on Microsoft Sentinel and EDR/XDR, engineer identity and Zero Trust access with Entra ID and Cloudflare, and automate the boring parts with PowerShell, Python, and Terraform. I like turning security work into code you can review and controls you can actually prove.

I write here to reinforce my own learning and to share what I figure out along the way — detection engineering, Kubernetes, Azure and Intune, Linux, infrastructure as code, and the note-taking and study habits (Obsidian, spaced repetition, neovim) that keep it all straight. If a post helps you skip a wall I already hit, even better.
